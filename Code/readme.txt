Guide d'Utilisation du système de gestion des réservations

Le système comporte deux vues, chacunes représentées par une classe de contrôle 
contenant toutes les méthodes requises : la vue client (Client.java) et la vue administrateur (Administrateur.java).

Pour les fins du projet, la classe SytemeGestionReservation.java implémente un outil par ligne de commande permettant
de tester les deux vues et leurs fonctionnalités.

Étapes d'utilisation :
1. Lancer l'application.
2. Le menu principal s'affiche.
3. Entrer le numéro de la vue désirée (1 pour Administrateur, 2 pour Client).
4. Entrer le numéro de l'option du sous-menu désiré suivant les options affichées.
5. Entrer les informations suivant le formulaire à l'écran.
6. Un message de confirmation sera affiché.
7. De retour au menu principal, répéter les étapes 3. à 6. pour effectuer une autre commande, ou choisir l'option 3 du menu pour quitter.
